# The Tortoise and the Hare - React Static Page

## Description:
This project is a simple static React page that tells the classic fable The Tortoise and the Hare.

## Features:
- Displays the story's title, summary, and moral
- Uses React components for a modular structure
- Styled using CSS for a clean layout

## Installation:
1. Clone this repository:  
git clone https://github.com/YOUR_GITHUB_USERNAME/tortoise-and-hare-react.git

2. Navigate into the project folder:  
cd tortoise-and-hare-react

3. Install dependencies:  
npm install

4. Start the development server:  
npm start
